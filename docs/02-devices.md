### 老人能用吗？对比传统开关

```
  ⚠️ 家里有老人，最怕的就是：「这个灯怎么开？」

  ┌──────────────────────────────────────────────────────────────┐
  │                  领普智能开关 vs 传统开关                       │
  ├──────────────┬───────────────────┬───────────────────────────┤
  │ 对比项        │ 传统开关           │ 领普智能开关               │
  ├──────────────┼───────────────────┼───────────────────────────┤
  │ 物理按键      │ 按下开/关 ✅       │ 按下开/关 ✅ (一模一样)    │
  │ 按键手感      │ 机械翘板/弹簧      │ 轻触面板，回弹清脆         │
  │ 外观          │ 白色塑料翘板       │ 白色平面面板，更简洁        │
  │ 开关大小      │ 标准 86 型         │ 标准 86 型（完全一样大）    │
  │ 需要联网吗    │ 不需要             │ 不需要（按键永远可用）      │
  │ 断电后       │ 正常用             │ 正常用（没电谁都不行）      │
  │ 学习成本      │ 零                │ 零（按就是了，没区别）      │
  └──────────────┴───────────────────┴───────────────────────────┘

  关键结论：
  ┌────────────────────────────────────────────────────────────┐
  │                                                            │
  │  对老人来说，智能开关 = 传统开关                               │
  │                                                            │
  │  • 按一下开灯，再按一下关灯 → 和以前完全一样                   │
  │  • 不需要手机，不需要联网，不需要学任何东西                     │
  │  • 唯一区别：年轻人还能用手机/语音控制，老人完全无感            │
  │                                                            │
  │  ✗ 不推荐的做法：                                            │
  │    用智能灯泡替代智能开关 → 老人按传统开关会断电，灯泡离线       │
  │    用触摸屏开关 → 老人可能不习惯，找不到按键位置               │
  │                                                            │
  │  ✓ 领普开关的优势：                                          │
  │    实体按键（不是触摸屏），手感明确                             │
  │    有指示灯（黑暗中也能找到开关位置）                          │
  │    按键面积大，老人不容易按错                                  │
  │                                                            │
  └────────────────────────────────────────────────────────────┘
```

## 三、中枢网关

### 为什么需要网关？

```
                    没有网关                          有网关
              ┌─────────────────┐            ┌─────────────────┐
              │                 │            │                 │
  手机蓝牙 ──▶│  只能在家控制     │  小爱音箱 ──▶│  远程控制 ✅      │
              │  不能定时 ✗      │  (网关)     │  定时自动化 ✅    │
              │  不能自动化 ✗    │            │  语音控制 ✅      │
              │  不能语音 ✗      │            │  智能场景 ✅      │
              │                 │            │                 │
              └─────────────────┘            └─────────────────┘
```

### 网关方案对比（选一个就行）

```
┌──────────────────┬──────────┬──────────────────────────────────┐
│ 方案              │ 参考价    │ 特点                             │
├──────────────────┼──────────┼──────────────────────────────────┤
│ ⭐ 小爱音箱 Pro    │ ～200 元  │ 蓝牙Mesh网关 + 语音控制，一举两得  │
│   小米 Sound      │ ～200 元  │ 同上，音质更好                    │
│   小米蓝牙Mesh网关 │ ～60 元   │ 最便宜，但没有语音功能             │
│   小米中枢网关 2   │ ～200 元  │ Mesh+Zigbee，扩展性强但无音箱     │
└──────────────────┴──────────┴──────────────────────────────────┘

  推荐 → 小爱音箱 Pro
  理由 → 你迟早要买音箱做语音控制，不如直接买带网关功能的，省一笔钱
```

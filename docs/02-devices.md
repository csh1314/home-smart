### 单火版 vs 零火版：怎么选？

```
  什么是火线和零线？

  配电箱
  ┌──────┐
  │      │── 火线(红) ──→ 开关 ──→ 灯 ──┐
  │      │                              │
  │      │── 零线(蓝) ────────────────────┘
  └──────┘

  火线：有电的线（电笔测试会亮）
  零线：回路的线（电笔测试不亮）
  灯控线：从开关到灯具的线（开关控制它通断来开关灯）
```

```
┌──────────────────────────────────────────────────────────────┐
│                  单火版 vs 零火版 对比                          │
├──────────────┬─────────────────────┬─────────────────────────┤
│              │ 零火版（零线版）      │ 单火版（单火线版）        │
├──────────────┼─────────────────────┼─────────────────────────┤
│ 底盒需要的线  │ 火线 + 零线 + 灯控线 │ 只要火线 + 灯控线        │
│ 价格          │ ～40-60 元          │ ～50-70 元（贵 10 元）   │
│ 稳定性        │ ★★★★★              │ ★★★★☆                  │
│ LED 灯兼容    │ 完美兼容             │ 低功率 LED 可能闪烁      │
│ 发热          │ 几乎不发热           │ 轻微发热（正常）          │
│ 待机功耗      │ 极低                 │ 略高（需要窃电维持工作）   │
├──────────────┼─────────────────────┼─────────────────────────┤
│ 适用场景      │ 新装修 / 有零线的老房 │ 老房子底盒没零线          │
└──────────────┴─────────────────────┴─────────────────────────┘

  ⚠️ 单火版为什么可能闪灯？
  ┌──────────────────────────────────────────────────────────┐
  │                                                          │
  │  单火开关没有零线，关灯后需要通过灯具「偷电」维持自身工作     │
  │  → 微弱电流流过灯具 → LED 灯敏感 → 可能出现微弱闪烁        │
  │  → 灯泡功率 < 5W 时特别明显                                │
  │                                                          │
  │  零火开关有独立的零线回路，关灯后电流不经过灯具               │
  │  → 彻底断开灯具 → 不可能闪灯                               │
  │                                                          │
  └──────────────────────────────────────────────────────────┘

  你家怎么选？
  ┌──────────────────────────────────────────────────────────┐
  │                                                          │
  │  你家是新装修，电工已预留零线 → 全部选零火版 ✅              │
  │                                                          │
  │  零火版更稳定、更便宜、不闪灯，没有任何理由选单火版          │
  │  唯一选单火版的情况：老房改造，底盒里确实没有零线             │
  │                                                          │
  └──────────────────────────────────────────────────────────┘
```

### 键数怎么选？（单键 / 双键 / 三键 / 四键）

```
  一个键控制一路灯。你家底盒里有几根灯控线，就选几键的开关。

  ┌──────────────────────────────────────────────────────────┐
  │                       选键数的原则                          │
  │                                                          │
  │  底盒里有 1 根灯控线 → 单键（控制 1 个灯/灯组）             │
  │  底盒里有 2 根灯控线 → 双键（控制 2 个灯/灯组）             │
  │  底盒里有 3 根灯控线 → 三键（控制 3 个灯/灯组）             │
  │  底盒里有 4 根灯控线 → 四键（少见，灯路特别多才用）          │
  │                                                          │
  │  ⚠️ 不要多买键数！                                        │
  │  底盒里只有 1 根灯控线，买三键开关也只能用 1 个键            │
  │  多出来的键按了没反应（除非当无线场景按钮用）                │
  │                                                          │
  └──────────────────────────────────────────────────────────┘

  怎么判断底盒里有几根灯控线？
  ┌──────────────────────────────────────────────────────────┐
  │                                                          │
  │  打开底盒面板，数一下线：                                   │
  │                                                          │
  │  • 红色 1 根 ─── 火线（所有开关共用）                       │
  │  • 蓝色 1 根 ─── 零线（所有开关共用）                       │
  │  • 其他颜色 ─── 灯控线（有几根就是几路灯）                  │
  │                                                          │
  │  例：红1根 + 蓝1根 + 绿1根黄1根 = 2根灯控线 → 选双键       │
  │  例：红1根 + 蓝1根 + 绿1根 = 1根灯控线 → 选单键             │
  │                                                          │
  │  💡 也可以看原来的旧开关是几键的                            │
  │     旧开关几键 → 新开关就选几键（最简单的判断方式）          │
  │                                                          │
  └──────────────────────────────────────────────────────────┘
```

```
  你家各位置的键数规划：

  ┌──────────┬──────┬──────────────────────────────────┐
  │ 位置      │ 键数  │ 原因                              │
  ├──────────┼──────┼──────────────────────────────────┤
  │ 客厅      │ 三键  │ 主灯 + 射灯 + 灯带，3 路灯        │
  │ 餐厅      │ 单键  │ 只有 1 个餐厅灯                   │
  │ 主卧      │ 双键  │ 主灯 + 氛围灯，2 路灯             │
  │ 卧室A     │ 双键  │ 主灯 + 辅灯，2 路灯               │
  │ 卧室B     │ 双键  │ 主灯 + 辅灯，2 路灯               │
  │ 厨房      │ 单键  │ 只有 1 个厨房灯                   │
  │ 公卫      │ 双键  │ 顶灯 + 镜前灯，2 路灯             │
  │ 主卫      │ 单键  │ 只有 1 个顶灯                     │
  │ 走廊      │ 单键  │ 只有 1 个走廊灯                   │
  │ 阳台      │ 单键  │ 只有 1 个阳台灯                   │
  └──────────┴──────┴──────────────────────────────────┘

  ⚠️ 以上是按常见户型规划的，实际以你家底盒里的灯控线数量为准！
     装修时让电工确认每个底盒有几路灯控线，按实际选键数。
```

### 老人能用吗？对比传统开关

```
  ⚠️ 家里有老人，最怕的就是：「这个灯怎么开？」

  ┌──────────────────────────────────────────────────────────────┐
  │                  领普智能开关 vs 传统开关                       │
  ├──────────────┬───────────────────┬───────────────────────────┤
  │ 对比项        │ 传统开关           │ 领普智能开关               │
  ├──────────────┼───────────────────┼───────────────────────────┤
  │ 物理按键      │ 按下开/关 ✅       │ 按下开/关 ✅ (一模一样)    │
  │ 按键手感      │ 机械翘板/弹簧      │ 轻触面板，回弹清脆         │
  │ 外观          │ 白色塑料翘板       │ 白色平面面板，更简洁        │
  │ 开关大小      │ 标准 86 型         │ 标准 86 型（完全一样大）    │
  │ 需要联网吗    │ 不需要             │ 不需要（按键永远可用）      │
  │ 断电后       │ 正常用             │ 正常用（没电谁都不行）      │
  │ 学习成本      │ 零                │ 零（按就是了，没区别）      │
  └──────────────┴───────────────────┴───────────────────────────┘

  关键结论：
  ┌────────────────────────────────────────────────────────────┐
  │                                                            │
  │  对老人来说，智能开关 = 传统开关                               │
  │                                                            │
  │  • 按一下开灯，再按一下关灯 → 和以前完全一样                   │
  │  • 不需要手机，不需要联网，不需要学任何东西                     │
  │  • 唯一区别：年轻人还能用手机/语音控制，老人完全无感            │
  │                                                            │
  │  ✗ 不推荐的做法：                                            │
  │    用智能灯泡替代智能开关 → 老人按传统开关会断电，灯泡离线       │
  │    用触摸屏开关 → 老人可能不习惯，找不到按键位置               │
  │                                                            │
  │  ✓ 领普开关的优势：                                          │
  │    实体按键（不是触摸屏），手感明确                             │
  │    有指示灯（黑暗中也能找到开关位置）                          │
  │    按键面积大，老人不容易按错                                  │
  │                                                            │
  └────────────────────────────────────────────────────────────┘
```

```
                    没有网关                          有网关
              ┌─────────────────┐            ┌─────────────────┐
              │                 │            │                 │
```

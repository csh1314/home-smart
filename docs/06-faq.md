# 06 - 常见问题与故障排查

## 故障排查

### 问题1：开关装好后灯闪烁

```
  原因排查流程：

  灯闪烁？
    │
    ├──▶ 零线是否接好？
    │     │
    │     ├── 没接 → 断电，检查 N 端子，重新插紧
    │     └── 接了 → 检查线头是否氧化，用剥线钳剥出新铜丝
    │
    └──▶ 灯具功率是否太低？
          │
          └── LED 灯 < 5W 可能会闪
              → 换个功率大一点的灯泡（≥7W）
              → 或者在灯具端并联一个消闪器（淘宝 ～10 元）
```

### 问题2：开关搜不到 / 配对失败

```
  ┌─────────────────────────────────────────────┐
  │  配对失败排查步骤：                            │
  │                                             │
  │  1. 手机蓝牙是否打开？        → 打开蓝牙      │
  │  2. 手机是否靠近开关？        → 走到 1 米内    │
  │  3. 开关是否在配对模式？                       │
  │     → 长按任意键 10 秒，指示灯快闪 = 配对模式  │
  │  4. 米家 App 版本是否最新？   → 更新 App       │
  │  5. 是否已被其他账号绑定？                      │
  │     → 长按 15 秒恢复出厂 → 重新配对            │
  │                                             │
  └─────────────────────────────────────────────┘
```

### 问题3：离家后无法远程控制

```
  远程控制不了？
    │
    ├── 小爱音箱是否在线？
    │    └── 检查音箱是否通电 + Wi-Fi 是否正常
    │
    ├── 音箱和开关是否在同一个米家账号下？
    │    └── 米家 App 检查设备列表
    │
    └── 路由器是否限制了设备联网？
         └── 检查路由器是否开了设备隔离或 MAC 过滤
```

### 问题4：窗帘电机行程不准

```
  窗帘开不到头 / 关不严？
    │
    └── 米家 App → 窗帘设备 → 右上角「...」
        → 行程设置 → 重新校准
        → 手动走一遍全开 → 全关 → 保存
```

### 问题5：开关偶尔不响应

```
  偶尔按了没反应？
    │
    ├── 蓝牙 Mesh 信号弱（设备离网关太远，中间没有中继）
    │    → 确保开关之间有 Mesh 中继链路
    │    → 实在不行在中间位置加一个领普开关做中继（开关本身就是中继节点）
    │
    └── 手机操作无响应但物理按键正常？
         → 网络问题，检查 Wi-Fi 和小爱音箱状态
         → 物理按键永远可用，不依赖网络
```

### 问题6：Wi-Fi 信号不好，会影响开关吗？

```
  ┌──────────────────────────────────────────────────────────┐
  │  短回答：物理按键完全不受影响，放心用                        │
  │                                                          │
  │  详细说明：                                                │
  │                                                          │
  │  开关用的是蓝牙 Mesh，不是 Wi-Fi                           │
  │  按开关 → 蓝牙信号直接控制灯 → 跟 Wi-Fi 没有任何关系        │
  │                                                          │
  │  Wi-Fi 只在以下场景才需要：                                 │
  │  • 出门后用 App 远程开关灯                                 │
  │  • 用小爱同学语音控制                                      │
  │  • 定时/自动化场景                                         │
  │                                                          │
  │  如果你担心语音/远程不稳定：                                 │
  │  → 你家有客厅和书房 2 个网线口，刚好部署 Mesh 路由          │
  │  → 主路由放客厅（网线口旁），子路由放书房（有线回程）        │
  │  → 推荐小米 Mesh 路由 AX3000 两只装（~300元）              │
  │  → 小爱音箱用 2.4G 连接（穿墙好）                          │
  │  → 路由器别开「设备隔离」功能                               │
  │                                                          │
  │  详见 → 01-architecture.md「Wi-Fi 不稳定会影响开关吗」章节  │
  └──────────────────────────────────────────────────────────┘
```

### 问题7：家里有老人，能用智能开关吗？

```
  ┌──────────────────────────────────────────────────────────┐
  │  完全可以，老人甚至感觉不到区别                              │
  │                                                          │
  │  • 按一下开灯、再按一下关灯 → 和传统开关一模一样             │
  │  • 不需要手机、不需要联网、不需要学任何东西                   │
  │  • 领普开关是实体按键（不是触摸屏），按感明确                 │
  │  • 有 LED 指示灯，黑暗中也能看到开关位置                    │
  │                                                          │
  │  ⚠️ 不要这样做：                                          │
  │  • 不要用智能灯泡 → 老人按传统开关会切断电源，灯泡离线       │
  │  • 不要用触摸屏开关 → 老人可能找不到按键位置                │
  │  • 不要禁用物理按键 → 某些「全智能」方案会这样做，很坑       │
  │                                                          │
  │  领普开关 = 传统开关 + 额外的智能功能                       │
  │  老人用传统方式，年轻人用手机/语音，互不干扰                  │
  │                                                          │
  │  详见 → 02-devices.md「老人能用吗？对比传统开关」章节        │
  └──────────────────────────────────────────────────────────┘
```

## 扩展建议

### 后续推荐设备（住进去再考虑）

```
┌─────────────┬──────────┬──────────────────────────────────┐
│ 设备         │ 参考价    │ 场景                              │
├─────────────┼──────────┼──────────────────────────────────┤
│ 领普无线开关  │ ～30 元   │ 贴床头，躺着关灯                   │
│ 人体传感器   │ ～50 元   │ 走廊/卫生间：人来灯开，人走灯灭     │
│ 温湿度传感器 │ ～40 元   │ 联动空调/除湿机                    │
│ 门窗传感器   │ ～40 元   │ 开门自动开灯                       │
│ NFC 贴纸    │ ～1 元    │ 碰一下触发场景                     │
└─────────────┴──────────┴──────────────────────────────────┘
```

### 后续扩展需要网线吗？

```
  ┌────────────────────────────────────────────────────────────┐
  │  你家只有客厅和书房有网线口，其他房间没有                       │
  │  后续加设备时，哪些需要 / 不需要网线？                         │
  │                                                            │
  │  不需要网线（蓝牙 Mesh / Wi-Fi）：                           │
  │  • 领普无线开关 ──── 蓝牙 Mesh ✅                           │
  │  • 人体传感器 ─────── 蓝牙 Mesh ✅                           │
  │  • 温湿度传感器 ───── 蓝牙 Mesh ✅                           │
  │  • 门窗传感器 ─────── 蓝牙 Mesh ✅                           │
  │  • NFC 贴纸 ────────── 不需要网络 ✅                          │
  │  • 小米摄像头 ─────── Wi-Fi ✅（但网线连接更稳定）            │
  │  • 小爱音箱 mini ─── Wi-Fi ✅                               │
  │                                                            │
  │  最好有网线（没有也能用 Wi-Fi 凑合）：                        │
  │  • NAS / 家庭服务器 ── Wi-Fi 太慢，需要有线连接              │
  │  • 有线 PoE 摄像头 ── 更稳定、不用单独供电                   │
  │  • Home Assistant ─── 推荐有线连接保障稳定                   │
  │  → 这些都放客厅或书房（有网线口的地方）就行                   │
  │                                                            │
  │  结论：米家生态的设备基本都是无线的，网线口少不影响扩展         │
  │                                                            │
  └────────────────────────────────────────────────────────────┘
```

### 何时考虑 Home Assistant？

```
  现在 ✗ ─── 设备少于 20 个，米家完全够用
    │
    │  住了半年到一年...
    │
  将来 ✓ ─── 当你觉得米家自动化不够灵活的时候
              比如：跨品牌联动、复杂条件判断、本地化运行
              那时候再折腾 Home Assistant 不迟
```

## 省钱技巧

```
┌──────────────────────────────────────────────────────────┐
│                       省钱指南                            │
├──────────────────────────────────────────────────────────┤
│                                                          │
│  1. 大促囤货                                              │

│  2. 拼多多百亿补贴                                        │

│  5. 传感器别急着买                                         │
│     住进去体验 1 个月再决定哪里需要                          │
│                                                          │

│     领普开关用标准 86 底盒，不用额外花钱买底盒               │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

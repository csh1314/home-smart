# 05 - 场景配置

## 第一步：整理房间分组

```
  米家 App → 首页 → 右上角「编辑」→ 管理房间

  建议分组：
  ┌──────────────────────────────────────────┐
  │  客厅     │ 客厅三键开关                  │
  │  餐厅     │ 餐厅单键开关                  │
  │  主卧     │ 主卧双键开关、主卧窗帘电机     │
  │  卧室A    │ 卧室A双键开关                 │
  │  卧室B    │ 卧室B双键开关                 │
  │  厨房     │ 厨房单键开关                  │
  │  公卫     │ 公卫双键开关                  │
  │  主卫     │ 主卫单键开关                  │
  │  走廊     │ 走廊单键开关                  │
  │  阳台     │ 阳台单键开关、阳台窗帘电机     │
  └──────────────────────────────────────────┘
```

## 第二步：给按键命名

```
  ⚠️ 这一步非常重要！命名后语音控制才精准

  客厅开关：
    键1 → 「客厅主灯」
    键2 → 「客厅射灯」
    键3 → 「客厅灯带」

  主卧开关：
    键1 → 「主卧灯」
    键2 → 「主卧氛围灯」

  卧室A开关：
    键1 → 「卧室A灯」      （或用孩子名/功能命名，如「书房灯」）
    键2 → 「卧室A辅灯」

  卧室B开关：
    键1 → 「卧室B灯」
    键2 → 「卧室B辅灯」

  公卫开关：
    键1 → 「公卫顶灯」
    键2 → 「公卫镜前灯」

  主卫开关：
    键1 → 「主卫灯」

  窗帘：
    → 「阳台窗帘」「主卧窗帘」

  设置路径：
  米家 App → 设备 → 右上角「...」→ 设备名称/按键名称
```

## 第三步：创建智能场景

### 场景一览

```
┌────────────┬──────────────────┬────────────────────────────────┐
│ 场景        │ 触发方式          │ 执行动作                        │
├────────────┼──────────────────┼────────────────────────────────┤
│ 回家模式    │ 手动 / NFC       │ 走廊灯开 + 客厅灯开 + 阳台窗帘开 │
│ 离家模式    │ 手动             │ 全屋灯关 + 全部窗帘关            │
│ 晚安模式    │ 语音 / 定时23:00 │ 全屋灯关 + 主卧窗帘关            │
│ 起床模式    │ 定时 7:30        │ 主卧窗帘开 + 阳台窗帘开          │
│ 观影模式    │ 语音             │ 客厅主灯关 + 射灯开 + 阳台窗帘关 │
│ 全关        │ 语音             │ 所有灯关闭                      │
└────────────┴──────────────────┴────────────────────────────────┘
```

### 创建场景步骤（以「晚安模式」为例）

```
  米家 App 操作流程：

  ┌─────────────┐
  │  智能 Tab    │
  │  右上角「+」  │
  └──────┬──────┘
         ▼
  ┌─────────────┐     ┌─────────────┐
  │ 选择触发方式  │ ──▶ │  手动执行    │  ← 选这个
  │             │     │  定时        │  ← 或者选定时 23:00
  └─────────────┘     └──────┬──────┘
                             ▼
  ┌──────────────────────────────────────┐
  │  添加执行动作（逐个添加）：              │
  │                                      │
  │  + 客厅主灯 ─────── 关闭              │
  │  + 客厅射灯 ─────── 关闭              │
  │  + 客厅灯带 ─────── 关闭              │
  │  + 餐厅灯 ────────── 关闭              │
  │  + 主卧灯 ────────── 关闭              │
  │  + 主卧氛围灯 ────── 关闭              │
  │  + 走廊灯 ────────── 关闭              │
  │  + ... （所有灯都关）                   │
  │  + 主卧窗帘 ─────── 关闭              │
  │                                      │
  └──────────────────────┬───────────────┘
                         ▼
  ┌─────────────────────────┐
  │  命名为「晚安模式」       │
  │  保存 ✅                 │
  └─────────────────────────┘
```

## 第四步：语音控制

### 常用语音指令

```
┌─────────────────────────────────────────────────────┐
│                  语音控制速查                          │
├─────────────────────────────────────────────────────┤
│                                                     │
│  单个设备控制：                                       │
│  「小爱同学，打开客厅主灯」                            │
│  「小爱同学，关闭主卧灯」                              │
│  「小爱同学，打开阳台窗帘」                            │
│  「小爱同学，阳台窗帘开到一半」    ← 支持百分比         │
│                                                     │
│  房间级控制：                                         │
│  「小爱同学，打开客厅的灯」       ← 客厅所有灯         │
│  「小爱同学，关掉卧室的灯」                            │
│                                                     │
│  全屋控制：                                          │
│  「小爱同学，关闭所有灯」                              │
│                                                     │
│  场景控制：                                          │
│  「小爱同学，晚安」              ← 自动匹配晚安模式    │
│  「小爱同学，执行回家模式」                            │
│  「小爱同学，我要看电影」         ← 匹配观影模式       │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### NFC 贴纸（可选进阶）

```
  成本：NFC 贴纸约 1 元/个

  用法：
  ┌─────┐
  │ NFC │ ← 贴在入户门旁边
  └──┬──┘
     │  手机碰一下
     ▼
  触发「回家模式」→ 走廊灯开 + 客厅灯开 + 阳台窗帘开

  设置：米家 App → 智能 → 新建场景 → NFC 触发
  把 NFC 贴纸贴在你每天进门会经过的位置
```

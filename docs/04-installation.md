### 确认底盒接线

打开每个底盒的盖板，确认线材：

```
  86 底盒内部示意（从正面看）：

  ┌───────────────────────────┐
  │                           │
  │    🔴 火线 (L)             │ ← 红色，1 根
  │                           │
  │    🔵 零线 (N)             │ ← 蓝色，1 根（你已预留 ✅）
  │                           │
  │    🟢 灯控线1 (L1)         │ ← 绿/黄/其他色，第一路灯
  │    🟡 灯控线2 (L2)         │ ← 第二路灯（双键及以上才有）
  │    ⚪ 灯控线3 (L3)         │ ← 第三路灯（三键才有）
  │                           │
  └───────────────────────────┘

  ⚠️ 如果分不清哪根是火线哪根是零线：
  → 用电笔测试：亮的是火线，不亮的是零线
  → 不确定就找电工帮忙，安全第一
```

### 窗帘盒检查

```
  窗帘盒截面（从侧面看）：

  ┌──────────────────────────────────────┐ 顶板
  │                                      │
  │  电源线入口 ──→ ┃                     │
  │  (一侧预埋)     ┃                     │
  │                 ┃   (安装轨道的空间)    │
  │                                      │
  └──────────────────────────────────────┘

  需要确认：
  ✅ 窗帘盒一侧有预埋的电源线（火线+零线）
  ✅ 窗帘盒宽度 ≥ 12cm（能放下电机）
  ❌ 如果没有电源线 → 让电工加一路，这是唯一额外布线
```

### 接线错误警告

```
  ┌─────────────────────────────────────────────────────────────┐
  │                                                             │
  │   ❌ 常见致命错误（可能烧开关甚至引发火灾）                     │
  │                                                             │
  │   错误1：火线零线接反                                         │
  │   ┌───────────────────────────────────────────┐             │
  │   │  L 端子 ← 接了蓝色零线  ← ❌ 烧开关       │             │
  │   │  N 端子 ← 接了红色火线  ← ❌ 触摸漏电风险   │             │
  │   └───────────────────────────────────────────┘             │
  │   正确做法：用电笔确认，亮的接 L，不亮的接 N                    │
  │                                                             │
  │   错误2：灯控线接到 N（零线）端子                               │
  │   ┌───────────────────────────────────────────┐             │
  │   │  灯控线接到了 N → 灯常亮，开关无法控制      │             │
  │   │  甚至短路跳闸                               │             │
  │   └───────────────────────────────────────────┘             │
  │   正确做法：灯控线只能接 L1/L2/L3                              │
  │                                                             │
  │   错误3：底盒塞线过多导致短路                                   │
  │   ┌───────────────────────────────────────────┐             │
  │   │  线头裸露部分互相接触 → 短路跳闸             │             │
  │   │  暴力塞入导致接线松动 → 接触不良/打火         │             │
  │   └───────────────────────────────────────────┘             │
  │   正确做法：线头拧紧插入端子，多余线缆整理折叠后再推入底盒       │
  │                                                             │
  │   错误4：带电操作                                              │
  │   ┌───────────────────────────────────────────┐             │
  │   │  没断空开就拆接线 → 触电！                   │             │
  │   └───────────────────────────────────────────┘             │
  │   正确做法：先去配电箱关闭对应回路，电笔验电后再操作              │
  │                                                             │
  └─────────────────────────────────────────────────────────────┘
```

### 电工布线 Checklist

```
  安装前请逐项确认（可截图给电工核对）：

  □ 每个底盒是否都有零线（蓝色）？
    → 没有零线 = 必须选单火版开关（贵约30元），或让电工补零线
  □ 每个底盒的火线是否只有一根？
    → 多根红线可能是串联到其他底盒的，不要随意拆断
  □ 灯控线数量是否和开关键数匹配？
    → 三键开关需要 3 根灯控线（L1/L2/L3）
    → 如果只有 2 根灯控线，只能装双键开关
  □ 底盒深度是否 ≥ 21mm？
    → T1/E3S/E3 Pro 底壳 21mm，老房子底盒如果太浅需要换深底盒
  □ 双控位置是否已确认处理方案？
    → 参考 07-switch-types.md 的方案一（有线+无线）
  □ 是否有串联底盒（一根火线穿过多个底盒）？
    → 串联底盒断电后需要确认所有串联位置一起处理

  给电工的沟通要点：
  ┌────────────────────────────────────────────────┐
  │  1. "所有底盒都需要预留零线"                      │
  │  2. "每个底盒内线头需要剥出约 8-10mm"             │
  │  3. "双控位置只保留主位（门口），副位（床头）弃用"  │
  │  4. "窗帘盒需要预留一路电源线（火线+零线）"        │
  │  5. "底盒深度不够的地方需要换加深底盒"              │
  └────────────────────────────────────────────────┘
```

## STEP 3.5：安装无线开关

> 无线开关不需要接线，纯粘贴安装，适合双控副位（床头）和补充控制位

### 适用场景

```
  ┌──────────────────────────────────────────────────────────┐
  │  无线开关用在哪？                                          │
  │                                                          │
  │  1. 双控副位（床头）── 替代传统双控的第二个开关              │
  │  2. 沙发旁 ── 躺沙发上也能关客厅灯                         │
  │  3. 走廊另一端 ── 走廊双控不用改线                         │
  │  4. 任何你觉得"这里也需要一个开关"的位置                    │
  │                                                          │
  │  领普无线开关型号：                                        │
  │  • K9B（单键） ～25元                                     │
  │  • K9B（双键） ～30元                                     │
  │  • 自发电，无需电池，按压发电 + 蓝牙 Mesh                  │
  │                                                          │
  └──────────────────────────────────────────────────────────┘
```

### 安装步骤

```
  ① 选择粘贴位置              ② 贴底座

  ┌──────────────┐           ┌──────────────┐
  │              │           │              │
  │   墙面/床头柜 │           │  ┌────────┐  │
  │              │           │  │ 3M胶底座│  │ ← 撕开3M胶贴纸
  │  ← 擦干净    │           │  │        │  │    用力按压10秒
  │              │           │  └────────┘  │
  └──────────────┘           └──────────────┘

  ③ 扣上开关面板              ④ 米家App配对

  ┌──────────────┐           ┌──────────────────────────────┐
  │  ┌────────┐  │           │  米家 App → + → 搜索无线开关  │
  │  │ 无线开关 │  │           │  配对后设置联动：              │
  │  │ (磁吸)  │  │ ← 啪嗒    │                              │
  │  └────────┘  │   扣上去   │  单击 → 切换xx灯开关状态      │
```

### 无线开关联动配置

```
  以主卧为例（门口有线开关 + 床头无线开关）：

  米家 App → 智能 → + 手动场景/自动化

  自动化1：
  ┌──────────────────────────────────────┐
  │  如果：无线开关「单击」               │
  │  就：主卧主灯 → 切换开关状态          │
  └──────────────────────────────────────┘

  自动化2（双键无线开关的第二键）：
  ┌──────────────────────────────────────┐
  │  如果：无线开关「第二键-单击」         │
  │  就：主卧辅灯 → 切换开关状态          │
  └──────────────────────────────────────┘

  自动化3（一键全关）：
  ┌──────────────────────────────────────┐
  │  如果：无线开关「长按」               │
  │  就：主卧主灯 → 关闭                 │
  │      主卧辅灯 → 关闭                 │
  │      主卧窗帘 → 关闭                 │
  └──────────────────────────────────────┘

  💡 无线开关的优势：支持单击/双击/长按三种触发
     一个双键开关最多可以绑定 6 个动作（2键 × 3种按法）
```

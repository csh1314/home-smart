## STEP 1：施工前准备

### 工具清单

```
  ┌─────────────────────────────────────────┐
  │             必备工具                      │
  │                                         │
  │  🔧 十字螺丝刀 ─── 拆装底盒面板          │
  │  🔧 一字螺丝刀 ─── 接线端子              │
  │  ⚡ 电笔/万用表 ─── 区分火线零线          │
  │  ✂️  剥线钳 ─────── 处理线头              │
  │  📱 手机 ────────── 米家 App 已安装       │
  │                                         │
  └─────────────────────────────────────────┘
```

### 确认底盒接线

打开每个底盒的盖板，确认线材：

```
  86 底盒内部示意（从正面看）：

  ┌───────────────────────────┐
  │                           │
  │    🔴 火线 (L)             │ ← 红色，1 根
  │                           │
  │    🔵 零线 (N)             │ ← 蓝色，1 根（你已预留 ✅）
  │                           │
  │    🟢 灯控线1 (L1)         │ ← 绿/黄/其他色，第一路灯
  │    🟡 灯控线2 (L2)         │ ← 第二路灯（双键及以上才有）
  │    ⚪ 灯控线3 (L3)         │ ← 第三路灯（三键才有）
  │                           │
  └───────────────────────────┘

  ⚠️ 如果分不清哪根是火线哪根是零线：
  → 用电笔测试：亮的是火线，不亮的是零线
  → 不确定就找电工帮忙，安全第一
```

### 窗帘盒检查

```
  窗帘盒截面（从侧面看）：

  ┌──────────────────────────────────────┐ 顶板
  │                                      │
  │  电源线入口 ──→ ┃                     │
  │  (一侧预埋)     ┃                     │
  │                 ┃   (安装轨道的空间)    │
  │                                      │
  └──────────────────────────────────────┘

  需要确认：
  ✅ 窗帘盒一侧有预埋的电源线（火线+零线）
  ✅ 窗帘盒宽度 ≥ 12cm（能放下电机）
  ❌ 如果没有电源线 → 让电工加一路，这是唯一额外布线
```

## STEP 2：安装中枢网关（小爱音箱）

耗时：约 10 分钟

```
  1. 选择位置（重要！）

     你家实际户型（参照 home.jpg）：
     ┌──────────────────────────────────────────────────┐
     │  入户↓                                           │
     │  ┌──────┐ ┌─────┐ ┌─────┐ ┌──────┐             │
     │  │餐厅  │ │厨房  │ │公卫  │ │卧室A │             │
     │  └──┬───┘ └─────┘ └──┬──┘ └──┬───┘             │
     │     │                 │  走廊  │                  │
     │  ┌──┴─────────────────┴───────┤  ┌────┐         │
     │  │       客厅                  │  │主卫│         │
     │  │  ⭐ Pro 放这(靠走廊侧)      │  └──┬─┘         │
     │  │                 ┌──────┐   │  ┌──┴──────┐    │
     │  │                 │卧室B │   │  │  主卧    │    │
     │  └──┬──────────────┴──────┘   │  └─────────┘    │
     │  ┌──┴───┐                     │                  │
     │  │阳台  │                     │                  │
     │  └──────┘                     │                  │
     └──────────────────────────────────────────────────┘

     ⭐ 小爱音箱 Pro（网关）→ 放客厅靠走廊一侧（蓝牙 Mesh 覆盖全屋）
     💡 小爱 mini（可选）  → 放主卧（纯语音，躺床上控制灯和窗帘）

  2. 插电 → 等待启动

  3. 手机操作：
     米家 App → 右上角「+」→ 搜索音箱型号 → 连接 Wi-Fi → 完成

  4. 验证成功：
     说「小爱同学，你好」→ 有回应 = 搞定
```

### 接线图解

**单键开关接线：**

```
  开关背面（领普 Q4 单键零线版）：

  ┌─────────────────────────┐
  │                         │
  │   ┌───┐         ┌───┐  │
  │   │ L │         │L1 │  │
  │   └─┬─┘         └─┬─┘  │
  │     │              │    │
  │   ┌───┐            │    │
  │   │ N │            │    │
  │   └─┬─┘            │    │
  │     │              │    │
  └─────│──────────────│────┘
        │              │
        │              │
  ┌─────│──────────────│────────────────────┐
  │     │              │         底盒内部    │
  │     │              │                    │
  │   ┌─┴──┐  ┌──┴──┐  ┌─────┐            │
  │   │ 红  │  │ 蓝  │  │ 绿  │            │
  │   │火线 │  │零线 │  │灯控线│            │
  │   └────┘  └────┘  └─────┘             │
  │                                        │
  └────────────────────────────────────────┘

  接法：
  L  ← 红色火线
  N  ← 蓝色零线
  L1 ← 绿色灯控线
```

**双键开关接线：**

```
  开关背面（领普 Q4 双键零线版）：

  ┌─────────────────────────────┐
  │                             │
  │   ┌───┐   ┌───┐   ┌───┐   │
  │   │ L │   │L1 │   │L2 │   │
  │   └─┬─┘   └─┬─┘   └─┬─┘   │
  │     │        │        │     │
  │   ┌───┐      │        │     │
  │   │ N │      │        │     │
  │   └─┬─┘      │        │     │
  └─────│────────│────────│─────┘
        │        │        │
  ┌─────│────────│────────│─────────────────┐
  │     │        │        │      底盒内部    │
  │   ┌─┴──┐  ┌─┴──┐  ┌─┴──┐  ┌─────┐    │
  │   │ 红  │  │ 蓝  │  │ 绿  │  │ 黄  │    │
  │   │火线 │  │零线 │  │灯1  │  │灯2  │    │
```

### 安装详细步骤

```
  ① 断电                          ② 拆旧开关
  ┌────────────┐                  ┌────────────┐
  │ 配电箱      │                  │  ┌──────┐  │
  │            │                  │  │ 旧开关 │  │
  │ [OFF] 照明 │ ← 关掉对应回路    │  │      │  │ ← 螺丝刀拆下面板
  │ [ON]  插座 │                  │  └──┬───┘  │    拔掉接线
  │ [ON]  空调 │                  │     │      │
  └────────────┘                  └─────│──────┘

  ③ 接新开关                       ④ 装回面板
  ┌────────────┐                  ┌────────────┐
  │            │                  │  ┌──────┐  │
  │  L ← 火线  │                  │  │ 新开关 │  │
  │  N ← 零线  │ ← 按端子标注接线  │  │ 💡   │  │ ← 装好，合上空开
  │  L1← 灯控  │                  │  │      │  │    指示灯亮起
  │            │                  │  └──────┘  │
  └────────────┘                  └────────────┘
```

### 配对入网

```
  开关通电后（指示灯亮起）：

  ┌──────────┐                    ┌──────────┐
  │  手机     │  ── 蓝牙 ──▶      │  开关     │
  │  米家App  │                    │  (待配对) │
  └────┬─────┘                    └──────────┘
       │
       ▼
  ┌─────────────────────────────────────────┐
  │  米家 App 操作步骤：                      │
  │                                         │
  │  1. 右上角「+」→ 自动发现新设备           │
  │  2. 点击领普开关 → 确认添加               │
  │  3. 选择房间（如「客厅」）                 │
  │  4. 给每个按键命名：                      │
  │     - 键1 → 「客厅主灯」                 │
  │     - 键2 → 「客厅射灯」                 │
  │     - 键3 → 「客厅灯带」                 │
  │  5. 按一下开关试试 → 灯亮了 = 成功 ✅     │
  │                                         │
  └─────────────────────────────────────────┘

  ⚠️ 搜不到设备？
  → 长按开关任意键 10 秒进入配对模式（指示灯快闪）
  → 再次搜索
```

### 安装顺序

```
  建议按以下顺序逐个房间安装（装一个配一个）：

  ① 客厅 ──▶ ② 走廊 ──▶ ③ 餐厅 ──▶ ④ 厨房 ──▶ ⑤ 公卫
     │
     └──▶ ⑥ 卧室A ──▶ ⑦ 卧室B ──▶ ⑧ 主卧 ──▶ ⑨ 主卫 ──▶ ⑩ 阳台

  💡 先装客厅（网关就在这）和走廊的开关
     客厅+走廊在房屋中间，作为 Mesh 中继
     让后面右侧卧室区和左侧阳台的设备配对更顺畅
```

## STEP 4：安装窗帘电机

> 强烈建议让商家上门安装轨道，省心省力

```

  Step 1：固定轨道

  │  ╔══════════════════════════════════════╗     │
  │  ║              电动轨道                ║     │
  │  ╚══════════════════════════════════════╝     │
  └──────────────────────────────────────────────┘

  Step 2：安装电机
  ┌──────────────────────────────────────────────┐
  │  ╔══════════════════════════════╦═══════╗     │

  Step 3：挂窗帘 + 配对
  ┌──────────────────────────────────────────────┐
  │  ╔══════════════════════════════╦═══════╗     │
  │  ║            轨道              ║ 电机  ║     │
  │  ╚═╤══╤══╤══╤══╤══╤══╤══╤══╤═╩═══════╝     │
```

### 配对与行程设置

```
  ┌──────────────────────────────────────────┐
  │  窗帘电机配对与校准：                      │
  │                                          │
  │  1. 米家 App →「+」→ 搜索窗帘电机         │
  │  2. 配对成功后进入行程设置                  │
  │                                          │
  │  行程校准过程：                            │
  │                                          │
  │  全开位置                    全关位置      │
  │  ◀═══════════════════════════════▶       │
  │  │←── 窗帘打开 ──────── 窗帘关闭 ──→│     │
  │                                          │
  │  ① App 点「打开」→ 窗帘走到最开 → 确认    │
  │  ② App 点「关闭」→ 窗帘走到最关 → 确认    │
  │  ③ 电机记住行程 → 以后自动停在正确位置     │
  │                                          │
  └──────────────────────────────────────────┘
```

## 安装进度 Checklist


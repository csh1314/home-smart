# 全屋轻智能落地教程

> 三室一厅一餐厅（1公卫+1主卫） | 米家生态 | 蓝牙 Mesh 2.0 | 低预算方案

## 项目概况

- **户型**：三室一厅一餐厅（客厅21.1㎡ / 主卧17.7㎡ / 卧室B 11.1㎡ / 卧室A 9.6㎡）
- **卫生间**：公卫 4.9㎡ + 主卫 5.0㎡
- **生态**：米家（小米）
- **协议**：蓝牙 Mesh 2.0
- **开关**：领普 LINPTECH E3 Pro（AG 肤感玻璃面板）
- **窗帘电机**：科创者 mini4（无刷直流，21dB 静音，10年质保）
- **网关**：Xiaomi 智能音箱 Pro（2025 年新款，Mesh 2.0 + 超级小爱）
- **范围**：灯光 + 窗帘 + 双控 + 夜灯
- **预算**：约 1,925 元基础版 / 1,990 元推荐版（含主卧语音覆盖）

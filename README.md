# 全屋轻智能落地教程

> 三室一厅一餐厅（1公卫+1主卫） | 米家生态 | 领普开关 | 低预算方案

## 项目概况

- **户型**：三室一厅一餐厅（客厅21.1㎡ / 主卧17.7㎡ / 卧室B 11.1㎡ / 卧室A 9.6㎡）
- **卫生间**：公卫 4.9㎡ + 主卫 5.0㎡
- **生态**：米家（小米）
- **开关**：领普 LINPTECH（蓝牙 Mesh）
- **范围**：灯光 + 窗帘（阳台+主卧电动窗帘）
- **预算**：约 1,560 元基础版 / 1,690 元推荐版（含语音覆盖）
